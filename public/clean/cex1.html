Clean & Clear Code

예시1 : 반복횟수선언


Bad Code

 
            for(let i = 0; i < 20;  i++){
                console.log("Seq of apple=" + i);  
            };

Good Code

            let NUMBER_OF_APPLE = 20; 

            for(let i = 0; i < NUMBER_OF_APPLE;  i++){
                console.log("Seq of apple=" + i);  
            };
